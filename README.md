# awesome-5g-omni

- 作成方針
  - 日本語で各種OSSやツール類の特徴・解説（使い方含）・簡易比較結果をまとめていきたい
  - ターゲット：
    - これからモバイルネットワークのOSSを開発しようとする開発者
    - これからモバイルネットワークのOSSを利用するユーザ
    - 既にOMNIに参加するメンバのノウハウ集
       - コンテンツの分類や、どこまで記載するか悩ましいですが、まずは記載してみて、少しづつ整理しながらコンテンツとして育てていければと考えています
